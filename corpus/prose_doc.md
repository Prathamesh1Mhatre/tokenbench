# Acme-UI CDN Deployment Guide

The Acme-UI component library publishes to a public GCS bucket. As of the latest release, the live version is v4.2.4 and the CDN base URL is https://cdn.example.com/acme-ui/dist. Cache TTL for immutable assets is set to 31536000 seconds (one year). Rollbacks are performed by repointing the 'latest' alias; the alias propagation SLA is 300 seconds. All uploads require the deploy service account acme-ui-cdn-deployer@proj.iam.gserviceaccount.com. Do not enable public write. Contact the Platform UI team for access.

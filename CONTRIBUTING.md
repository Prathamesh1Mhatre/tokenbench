Contributions welcome.

1. Add an adapter in adapters/__init__.py (one @adapter function).
2. Run: venv/bin/python tokenbench.py --tools yourtool --seeds 20
3. Commit the adapter + results/yourtool.json + a line in README scoreboard.

Rules: measured numbers only, N>=20 for library tools, note any lane where
the tool does not apply. Vendor claims stay out of results/.
